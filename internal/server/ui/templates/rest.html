Endpoints

  • GET /api/v1/items List items (query: limit, offset, sort, labels)
  • POST /api/v1/items Create item
  • GET /api/v1/items/{id} Get item (supports If-None-Match)
  • PUT /api/v1/items/{id} Upsert item (supports If-Match)
  • DELETE /api/v1/items/{id} Delete item
  • GET /health Health check

Examples

# List all items
curl http://localhost:{{.Port}}/api/v1/items

# Create an item
curl -X POST http://localhost:{{.Port}}/api/v1/items \
  -H 'Content-Type: application/json' \
  -d '{"name":"demo","data":{"hello":"world"},"labels":{"env":"dev"}}'

# List with label filter and sorting
curl 'http://localhost:{{.Port}}/api/v1/items?labels=env:dev&sort=name&limit=10'

# Conditional GET (returns 304 if unchanged)
curl -H 'If-None-Match: "abc123"' http://localhost:{{.Port}}/api/v1/items/ID

# Conditional update (fails with 412 if ETag mismatch)
curl -X PUT -H 'If-Match: "abc123"' \
  http://localhost:{{.Port}}/api/v1/items/ID \
  -d '{"name":"updated"}'
